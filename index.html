Fitness Tracker App

Android Development

This fitness tracking application helps users maintain their health goals by tracking workouts, nutrition, and progress. The app features real-time workout tracking, custom workout plans, progress analytics, and social sharing capabilities.

I developed this app using Kotlin and implemented MVVM architecture with Room database for local storage and Retrofit for API communication. The UI was designed following Material Design guidelines to ensure a seamless user experience.

Client FitLife Inc.
Timeline 3 Months
Technologies Kotlin, MVVM, Room, Retrofit
Role Lead Android Developer
